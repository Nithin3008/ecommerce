### Need For speed
---
An e-commerce project for purchasing cars

## Techonlogies Used
---
* React Js(useReducer,useContext)
* React Router 6
* Tailwind css
* react-toastify
* mockbee server
* Redux Toolkit
* Font awesome Icons

## Features
---
*  Home Page
*  Product Listing Page
*  Single Product image
*  Filters
     * Price
     * Category
     * Ratings
*  Wishlisth Management
*  Cart Management
*  Search
*  Loading & Alerts
*  Authentication  
    * Sign-up
    * Login
    * Logout
* Address Management
* Checkout
